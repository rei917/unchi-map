# v19 changes

## グループ記録の複数ピン表示改善

- `MapView.tsx` のピンずらし処理を強化しました。
- これまでは `lat.toFixed(6):lng.toFixed(6)` の完全一致に近い座標だけを同一地点として扱っていました。
- v19では、15m以内の記録を「同じ場所」としてクラスタ化し、全員分のピンを円状に展開します。
- DBに保存される `lat/lng` は変更しません。表示座標だけを調整します。
- マイ記録・グループ記録のどちらでも同じ処理が適用されます。

## 確認方法

1. 同じグループで複数ユーザーが同じ場所付近に記録する。
2. グループ画面を開く。
3. 全員分の🚽ピンが円状にずれて表示されることを確認する。
4. 各ピンを押して、正しい投稿者・コメントが表示されることを確認する。
